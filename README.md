# Web_Angular_Eltex

Репозиторий для выполнения домашних заданий курса **«Основы веб-разработки с Angular»** от компании **Eltex**.

## Описание

Проект включает в себя практические задания по веб-разработке, начиная с основ HTML и заканчивая разработкой приложений на Angular.

## Структура проекта
 
- `index.html` — главная страница (портфолио / блог веб-разработчика)
- `blog.html` — страница блога со списком статей
- `style.css` — стили для обеих страниц
 
## Технологии
 
- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, CSS-переменные, псевдоэлементы, анимации, адаптивность)
- Шрифт: Inter (Google Fonts)
